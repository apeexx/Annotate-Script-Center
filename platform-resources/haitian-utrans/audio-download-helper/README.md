# Haitian uTrans 音频下载助手资料

## 目标页

- 任务详情页：
  - `index.php?d=worker&c=work`

## 接口规律

- 当前音频接口形态：
  - `/index.php?d=worker&c=audio&m=audio_path&projectid=<...>&batchid=<...>&audioid=<...>`
- 当前已知特点：
  - 接口返回体本质上是 WAV 二进制
  - 服务端响应头不正确，浏览器直接打开时可能显示乱码文本

## 最小接入边界

- 只从当前页面上下文动态读取 `projectid / batchid / audioid`
- 优先读取：
  - 当前 URL
  - 页面隐藏字段
  - 页面已有 DOM 里的音频候选信息
- 扩展设置面板只提供一个最小开关：
  - 是否开启详情页悬浮下载按钮
- 当前实现只负责：
  - 挂悬浮下载按钮
  - 组装当前音频接口地址
  - 继承当前登录态下载
  - 通过扩展桥接脚本避开页面 CSP 对内联脚本的限制
  - 重新包装为 `audio/wav` Blob 并触发浏览器下载

## 当前不做

- 不做平台后端
- 不做独立服务
- 不做自动下载
- 不做批量下载
- 不做自动提交
- 不长期保存完整音频 URL

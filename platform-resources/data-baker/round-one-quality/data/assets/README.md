# DataBaker Data Assets

`platform-resources/data-baker/round-one-quality/data/assets/` 存放脚本级数据资产：

- `mappings/`：CSV 字段口径、旧字段兼容和唯一键规则说明
- `samples/`：脱敏后的 CSV / raw record / upload payload / latest meta / upload events 样例，供协作者核对上传、下载和 history 链路

这些文件只保留结构与占位示例，不包含真实用户数据、token、cookie、签名 URL 或可回放音频地址。

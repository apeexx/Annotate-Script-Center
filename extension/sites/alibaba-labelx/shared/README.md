# 阿里 LabelX Shared 模块

## 目录说明

- `audio-controller-core.js`：阿里 LabelX 通用音频控制核心，供快判和转写复用。
- `submit-actions.js`：阿里 LabelX 通用提交快捷键动作，仅负责查找并点击页面系统“提交”按钮。

## 通用能力范围

- 音量控制（含重置）
- 默认倍速应用与倍速步进
- 前进 / 后退步长
- 自动播放当前题音频
- 切题时停旧播新
- 单题音频播放互斥（避免混播）

## 不包含能力

- 快判雷题、差异视图、AI 建议
- 转写文本填入与文本处理
- 快判 `400` 条自定义每页请求逻辑
- 任何直接平台 API 调用、自动确认二次弹窗、自动领取和自动流转

## 复用方式

- 快判和转写各自保留 `audio-controller.js` 薄封装。
- 薄封装负责：
  - 读取脚本本地配置
  - 传入页面结构 profile（题卡选择器、当前题解析方式）
  - 对外保持原有 `globalThis` API 名称
